Objective 4:
Use software development processes to analyse a project problem, and to design, build and test a corresponding software solution.

Mettle RPG Assistant

Mettle RPG Assistant started as a simple problem: The tools that exist for managing table top role playing games either don't support more than one system or only support a limited set of systems. The team for Mettle set out to design and build a software solution that could handle any system by providing users with a universal set of tools and the ability to create the various sheets/forms that their chosen system uses.
